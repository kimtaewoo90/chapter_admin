# chapter_admin

Chapter 앱 관리자 페이지 (Flutter Web)

## 기능

- **결제상황** — 주문 목록, 입금 확인, PDF 다운로드
- **유저상황** — Firestore 유저 목록, 활동 현황, 관리자 메모
- **Analytics** — 준비 중

## 실행

```bash
flutter pub get
flutter run -d chrome
```
